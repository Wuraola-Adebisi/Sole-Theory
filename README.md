# SOLE THEORY

A premium footwear store front built with HTML and CSS.

## Features
- Fixed navigation header
- Full screen hero section
- Editorial alternating product layout
- Horizontal scroll featured strip
- CSS marquee ticker animation
- Split screen about section
- Horizontal journal cards
- Newsletter signup form
- Fully responsive — mobile, tablet, desktop
- Back to top button

## Built With
- HTML5
- CSS3 — Grid, Flexbox, Custom Properties, Animations
